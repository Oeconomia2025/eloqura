# Token Explorer

Analyze tokens, pools, and on-chain metrics.

**Source:** `client/src/pages/examine.tsx`

## Overview

The Examine page provides deep analytics for individual tokens and Eloqura pools. Users can search for tokens, view pool reserves, and track liquidity positions.

## Features

- **Token search** by name, symbol, or contract address
- **Pool filtering** and discovery
- **Reserve balances** for each pool
- **Liquidity position tracking** for connected wallet
- **Pool creation interface** for new token pairs
- **Token metadata** display (name, symbol, decimals, supply)
